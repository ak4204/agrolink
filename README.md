# AgroLink (Vite wrapper)

This repository contains only the `src/` folder originally. I added a minimal Vite-based project setup so you can run and build the app.

How to run

1. npm install
2. npm start

How to build

1. npm install
2. npm run build

Files added:
- `package.json` - project manifest and scripts
- `vite.config.js` - vite config with React plugin
- `public/index.html` - root HTML file
- `src/main.jsx` - entrypoint that renders `Layout` from `src/Layout.js`
- `src/index.css` - minimal CSS reset
- `.gitignore`, `README.md`
